Detailed Description

This module is used for computing pressures and transients using the Fast Nearfield Method (FNM)

Continuous Wave (CW) pressure fields and gradient are computed in frequency domain using an expression, which is equivalent to the Fourier transform of the Rayleigh integral. The procedure is in the literature referred to as the Fast Near Field Method [14] .
